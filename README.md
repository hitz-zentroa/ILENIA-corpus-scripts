## ILENIA corpus scripts

This repository contains the code, resources and configuration files developed and used
to download and process Basque corpora in the framework of the ILENIA project.

The contents are as follows:

* [aldizkariak](aldizkariak): repository to download and preprocess academic journals in Basque
* [berria](berria): repository to download and preprocess news pieces from BERRIA
* [boletines](boletines): repository to download and preprocess official journals from the Basque Country
* [parleus](parleus): repository to download and preprocess minutes of the Basque Parliament
* [dolma](dolma): configuration files to male the final compilation of the corpus
